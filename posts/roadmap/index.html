    Roadmap

    2021-04-09

    The current version of GritDB functionality has been around for 5 years, powering multiple (web) applications. It has always been open source. With the new name and renewed commitment to make the functionality available and ensure that data in an existing GritDB database is easily accessible outside of the complete running system comes a new roadmap for features.
